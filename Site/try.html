What loads

CopperOS v1.0 Copperhead boots directly from the ISO.

The frame points at the hosted v86 emulator with the CopperOS ISO already attached as boot media. If a browser or network blocks embedding, the full emulator button opens the same ready-to-boot session in a new tab.

Release profile

Classic startup into a desktop

Copperhead starts with a NASM boot sector, enters a freestanding 32-bit C kernel, draws staged boot screens, and then moves through setup into the desktop.

Inside the OS

What to expect after boot.

First-run setup

Create a username and 4-digit passcode, choose install or try mode, pick a theme, and select network settings.

Desktop apps

Open File Explorer, Internet Explorer, Activity Manager, Paint, Settings, Updates, Calculator, or TXT Editor.

Real limits

Internet Explorer accepts typed URLs, but real browsing still needs a full NIC, TCP/IP, DNS, TLS, and HTML stack.

CopperOS is experimental software. Browser emulation depends on browser support and release asset availability.
