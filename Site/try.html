Try CopperOS

The emulator on this page runs entirely in your browser and boots copperos.iso directly, so the Copperhead startup sequence begins as soon as you press Start.

What loads

CopperOS v1.0 Copperhead boots directly from the ISO.

This page runs the open-source v86 x86 emulator client-side and attaches copperos.iso as boot media. The kernel, BIOS, and ISO all download into your browser and run locally — nothing is sent to a server.

Release profile

Classic startup into a desktop

Copperhead starts with a NASM boot sector, enters a freestanding 32-bit C kernel, draws staged boot screens, and then moves through setup into the desktop.

Inside the OS

What to expect after boot.

First-run setup

Create a username and 4-digit passcode, choose install or try mode, pick a theme, and select network settings.

Desktop apps

Open File Explorer, Internet Explorer, Activity Manager, Paint, Settings, Updates, Calculator, or TXT Editor.

Real limits

Internet Explorer accepts typed URLs, but real browsing still needs a full NIC, TCP/IP, DNS, TLS, and HTML stack.

CopperOS is experimental software. Browser emulation depends on your browser supporting WebAssembly and on the release ISO being reachable.
