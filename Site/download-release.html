Build & Run (from source)

If you build from source, use the repository Makefile:

make
  # then run in QEMU
  make run
  # or headless serial check
  make run-headless

Controls

Setup uses a 4-digit passcode. On the VM:

  • I J K L: move pointer
  • Tab: cycle icons
  • Enter: click/open
  • S: open Start menu
  • Esc: close app
  • 1..8: quick-open apps
