# Built-In Apps

- `File Explorer`: shows the CopperOS project tree inspired by the reference image
- `Internet Explorer`: typed URL entry with honest real-network-stack status, no fake page-load simulation
- `Activity Manager`: system status, performance counters, and diagnostics
- `Paint`: a tiny keyboard-driven pixel canvas
- `System Settings`: Wi-Fi profile selection, theme selection, Start menu style, and diagnostics
- `System Updates`: checks the local release and reports that online update checks need real networking
- `Calculator`: performs basic arithmetic with keyboard input
- `TXT Editor`: a single-document text editor with in-memory auto-save status
